# CampusBuy

CampusBuy is an Android app that provides a platform for students to sell and buy second-hand items useful for campus life such as books, tech items, etc.

# Features
- Signup/Login system
  - Firebase authentication
- Add items to sell
- Mark item as interested
- Buy item for price given buy seller
- Chat with the seller to negotiate price / FAQ
